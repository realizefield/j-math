① いよいよ文章題！
📌 こんな問題を解きます
「連続する3つの自然数があります。
その真ん中の数の平方は，その他の2つの数の和より8大きい。
この3つの数を求めなさい。」

式が書かれてないので，自分で 方程式を立てる 必要がある！
② 大切な基礎知識
📌 用語の確認
① 自然数：$1, 2, 3, 4, \ldots$ のような 正の整数（個数を表す数）
② 整数：$\ldots, -3, -2, -1, 0, 1, 2, 3, \ldots$（マイナスや $0$ も含む）
③ 連続する3つの自然数：$3, 4, 5$ や $11, 12, 13$ のような数
④ 平方：2乗すること（例：$5$ の平方 $= 5^2 = 25$）
③ 連続する3つの数の表し方
📌 とても大事！
最初の数を $x$ とすると：
$x$，$x+1$，$x+2$
最初$x$
+1→
真ん中$x+1$
+1→
最後$x+2$
📌 なぜこう表す？
たとえば「$3, 4, 5$」と「$11, 12, 13$」も連続する3数。
どちらも「最初」「+1」「+2」の関係になっている！
だから，最初の数を $x$ で代表させて表す。
④ 文章 → 方程式 のコツ
💡 文章中の は や が をさがそう！　そこが「=（イコール）」になる！
📌 例の文章を分析
「真ん中の数の平方 は，他の2数の和より8大きい」

左側：真ん中の数の平方 = $(x+1)^2$
右側：他の2数の和より8大きい = $\{x + (x+2)\} + 8$
だから方程式は：
$(x+1)^2 = x + (x+2) + 8$
⑤ 解いていこう
📌 ステップ１：式を整理
$(x+1)^2 = x + (x+2) + 8$
左辺を展開：$x^2 + 2x + 1$
右辺を計算：$2x + 10$
$x^2 + 2x + 1 = 2x + 10$
全部左辺へ：$x^2 - 9 = 0$
📌 ステップ２：因数分解で解く
$x^2 - 9 = 0$ → $(x+3)(x-3) = 0$
$x = -3, 3$
⑥ 解の吟味（重要！）
二次方程式の解が 2つ出る けど，どちらも答えになるとは限らない！
📌 自然数という条件をチェック
$x = -3$ → 最初の数が $-3$？　自然数じゃない！　❌
$x = 3$ → 最初の数が $3$。3, 4, 5 ですべて自然数。✅
だから $x = 3$ が答え。
3つの数は $3, 4, 5$
「自然数」「正の整数」と書いてあったら，マイナスや$0$は捨てる！
⑦ 解き方の流れ
📌 文章題の手順（5ステップ）
1
連続3数を $x, x+1, x+2$ とおく
2
文章中の 「は」「が」 をさがして方程式を立てる
3
展開・整理して $ax^2 + bx + c = 0$ の形に
4
因数分解 or 解の公式で解く
5
条件に合わない解は捨てる（自然数か？など）
⑧ 例で確認しよう
例)真ん中の平方 = 他の2数の和 + 4 となる連続3つの自然数を求めよ
3数を文字で
$x, x+1, x+2$ とする
方程式
$(x+1)^2 = x + (x+2) + 4$
展開
$x^2 + 2x + 1 = 2x + 6$ → $x^2 = 5$
確認
$x = \pm\sqrt{5}$ → 自然数ではない！　答えなし
この場合，自然数の解はなし
↑のように，解はあっても自然数じゃないケースもある。条件チェックが大事！
📌 まとめ
  • 連続する3つの自然数は $x, x+1, x+2$ と表す
  • 文中の 「は」「が」 がイコールになる
  • 展開・整理して二次方程式に
  • 解いて 条件に合わない解は捨てる
  • 「自然数」「正の整数」 → マイナスや$0$ は不適
  • 「最も小さい数を求めよ」「3数を求めよ」など，問題文の指示に従って答える
