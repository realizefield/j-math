① 基本の公式
📌 重要公式（$a \geq 0$ のとき）
$$\sqrt{a^2} = a$$ ルートの中が「ある数の $2$ 乗」になっているときは，ルートを外せる！
例：$\sqrt{5^2} = \sqrt{25} = 5$　／　$\sqrt{3^2} = \sqrt{9} = 3$
② ⚠️ ここで問題が起きる！
❗ もし $a$ がマイナスだったら？
例として $\sqrt{(-2)^2}$ を計算してみよう。
もし「ルートを外すだけ」と考えると… $\sqrt{(-2)^2} = -2$？
でもそれは間違い！
📌 正しい計算の手順
① まず ( ) の中を $2$ 乗する　→　$(-2)^2 = 4$
② $\sqrt{4} = 2$
つまり　$\sqrt{(-2)^2} = 2$　（$-2$ ではない！）
ルート記号「$\sqrt{\phantom{x}}$」の答えは，かならずプラス（または0）になる！
③ 📖 絶対値とは？（新しい記号 $|\,\phantom{a}\,|$ を学ぼう）
📌 絶対値 $|a|$ ってなに？
平方根の公式を完全にするために，絶対値という新しい記号を学びます。
絶対値とは，ひとことで言うと：

👉 「0 からの距離」　のこと！
「$-2$」も「$2$」も，$0$ からの距離はどちらも $2$
つまり：
$|2| = 2$
$|-2| = 2$
$|5| = 5$
$|-5| = 5$
$|0| = 0$
$|-100| = 100$
👉 符号を取り除いた値（必ず $0$ 以上）と覚えよう！
イメージ：東京駅から「新宿方面に $3$ km」も「千葉方面に $3$ km」も，どちらも「$3$ km 離れている」のは同じ。方向は無視して「距離」だけを表すのが絶対値！
④ 絶対値を使って公式を完成！
📌 重要公式（$a$ がどんな値でも成り立つ）
$$\sqrt{a^2} = |a|$$ → $a$ がプラスなら $a$，マイナスなら $-a$（プラスに戻す）
📌 具体例で確認
$\sqrt{3^2} = |3| = 3$　（$a=3$ の場合）
$\sqrt{(-3)^2} = |-3| = 3$　（$a=-3$ の場合，結果は $3$！）
$\sqrt{7^2} = |7| = 7$
$\sqrt{(-7)^2} = |-7| = 7$
⑤ 例で確認しよう
例１）$\sqrt{6^2}$ を計算する
公式
$\sqrt{a^2} = a$（$a \geq 0$）を使う
計算
$\sqrt{6^2} = 6$
答え：$\mathbf{6}$
例２）$\sqrt{(-5)^2}$ を計算する（マイナスに注意！）
注意
ルートの中はまず $2$ 乗を計算！
手順①
$(-5)^2 = 25$
手順②
$\sqrt{25} = 5$
答え：$\mathbf{5}$　（$-5$ ではない！）
例３）$-\sqrt{8^2}$ を計算する（ルートの外にマイナス）
分解
$-\sqrt{8^2} = -(\sqrt{8^2})$　外のマイナスはあとで！
ルート部分
$\sqrt{8^2} = 8$
外のマイナス
$-8$
答え：$\mathbf{-8}$
例４）$\sqrt{(-3)^2}$ と $\sqrt{-3}$ の違い
左の式
$\sqrt{(-3)^2}$ : まず $2$ 乗 → $(-3)^2 = 9$ → $\sqrt{9} = 3$
右の式
$\sqrt{-3}$ : ルートの中がマイナス → 存在しない！
ポイント
「$2$ 乗してからルート」と「直接ルート」はまったく別物！
$\sqrt{(-3)^2} = 3$ ，$\sqrt{-3}$ は存在しない
📌 まとめ
  • $\sqrt{a^2} = a$（$a \geq 0$ のとき）
  • $\sqrt{a^2} = |a|$（$a$ がどんな値でも）
  • 絶対値 $|a|$ は「$0$ からの距離」＝「符号を取り除いた値」
  • ルート記号の中が「$2$ 乗の形」なら，先に $2$ 乗を計算してから √ を外す
  • ルートの値は必ず $0$ 以上（マイナスにならない）
  • $\sqrt{(-3)^2} \neq \sqrt{-3}$　に注意！
