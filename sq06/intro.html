① たし算・ひき算は要注意！
📌 SQ03で学んだこと
かけ算・わり算では $\sqrt{a} \times \sqrt{b} = \sqrt{ab}$ という便利な公式がありました。
でも，たし算・ひき算では同じようにはいきません！
$\sqrt{2} + \sqrt{3} = \sqrt{5}$ は 大きな間違い！　たし算は中の数を足してはいけない！
📌 値で確かめてみよう
$\sqrt{2} \approx 1.414$，$\sqrt{3} \approx 1.732$
$\sqrt{2} + \sqrt{3} \approx 1.414 + 1.732 = 3.146$
一方，$\sqrt{5} \approx 2.236$
全然違う値！　だから $\sqrt{2} + \sqrt{3} \neq \sqrt{5}$
② たし算・ひき算のルール
📌 公式
$a\sqrt{b} + c\sqrt{b} = (a+c)\sqrt{b}$
$a\sqrt{b} - c\sqrt{b} = (a-c)\sqrt{b}$
(ルートの中が同じ場合のみ)
📌 言葉でいうと
ルートの中が同じもの同士は，係数(ルートの外の数)だけたし算・ひき算する。
ルートの中の数は 変えない！
$2\sqrt{3} + 5\sqrt{3} = (2+5)\sqrt{3} = 7\sqrt{3}$
これは「同類項のまとめ方」と同じ考え方です。$2x + 5x = 7x$ となるのと一緒！
③ 文字式と比べてみよう
文字式
$2x + 5x$
$= 7x$
ルート
$2\sqrt{3} + 5\sqrt{3}$
$= 7\sqrt{3}$
📌 「ルート部分」が同じものだけまとめる
文字式で $2x + 3y$ がそれ以上計算できないように，
$2\sqrt{3} + 3\sqrt{5}$ もそれ以上計算できません。
❌ ダメな例
$\sqrt{2} + \sqrt{3} = \sqrt{5}$
中を足してしまっている
✅ 正しい
$\sqrt{2} + \sqrt{3}$
そのまま
中が違うのでまとめられない
④ 一見まとめられないように見えるが…
📌 たとえば $\sqrt{8} + \sqrt{2}$
パッと見ると，ルートの中の数が違う(8と2)のでまとめられないように見えます。
でも，SQ04で学んだ「ルートの中を簡単にする」を使うと…？
$\sqrt{8} = \sqrt{4 \times 2} = 2\sqrt{2}$
だから，$\sqrt{8} + \sqrt{2} = 2\sqrt{2} + \sqrt{2} = 3\sqrt{2}$
たし算・ひき算をする前に，まずルートの中を簡単にするのがコツ！
⑤ 例で確認しよう
例１)$\sqrt{5} + 3\sqrt{5}$ を計算する
確認
ルートの中はどちらも $5$ で同じ → まとめられる！
係数を見る
$\sqrt{5}$ の係数は $1$，$3\sqrt{5}$ の係数は $3$
係数を計算
$1 + 3 = 4$ なので $4\sqrt{5}$
$\sqrt{5} + 3\sqrt{5} = 4\sqrt{5}$
例２)$\sqrt{12} + \sqrt{3}$ を計算する
中を簡単に
$\sqrt{12} = 2\sqrt{3}$ (SQ04で学習)
書きかえる
$\sqrt{12} + \sqrt{3} = 2\sqrt{3} + \sqrt{3}$
係数を計算
$2 + 1 = 3$ なので $3\sqrt{3}$
$\sqrt{12} + \sqrt{3} = 3\sqrt{3}$
例３)$\sqrt{2} + \sqrt{7}$ を計算する
確認
$\sqrt{2}$ も $\sqrt{7}$ もルートの中が違う数
注意
これ以上簡単にもできないので…
結論
そのまま $\sqrt{2} + \sqrt{7}$ が答え！
$\sqrt{2} + \sqrt{7}$ (まとめられない)
📌 まとめ
  • たし算・ひき算は ルートの中が同じ場合のみ まとめられる
  • 公式：$a\sqrt{b} + c\sqrt{b} = (a+c)\sqrt{b}$
  • 「同類項のまとめ方」と同じ考え方
  • $\sqrt{2} + \sqrt{3} = \sqrt{5}$ は 大きな間違い！
  • 計算する前に，まずルートの中を簡単にする(SQ04)
  • 中が違ってまとめられない場合は，そのままが答え
