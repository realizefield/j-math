基本公式
📌 たして・かける の公式（展開①の逆）
χ²＋(a＋b)χ＋ab ＝ (χ＋a)(χ＋b)
かけて定数項c，たしてχの係数b になる2数 a, b を探す！
🤔 なぜ「かけて・たして」で見つかるの？
EF01の展開を逆向きに読むだけ！
1
EF01の展開：(χ＋a)(χ＋b) ＝ χ²＋(a＋b)χ＋ab
χの係数は a＋b（たした値），定数項は ab（かけた値）
2
逆に，「かけてc，たしてb」になる2数 a,b を探せばよい！
見つかれば → (χ＋a)(χ＋b) と書けばOK
3
2数の符号は定数項とχの係数の符号で決まる
→ 3つのパターンに分けて覚えよう！
まず定数項の符号を見てパターンを判断 → そのパターンのルールで2数を探す，の2ステップ！
3パターンの見分け方（符号早見表）
パターンχの係数b定数項c2数の符号形
パターン1 ＋＋両方＋(χ＋ )(χ＋ )
パターン2 －＋両方－(χ－ )(χ－ )
パターン3 ＋ or －－符号が逆(χ＋ )(χ－ )
各パターンの詳細と練習
パターン1(χ＋　)(χ＋　)　← 両プラス
χ²＋bχ＋c　（b＞0，c＞0）
ルール：かけてc，たしてb になる正の整数の組を探す
定数項がプラス・χの係数もプラス → 2数とも＋
例）χ²＋5χ＋6　→　2と3　→　(χ＋2)(χ＋3)
➡ パターン1 を練習する（例題・類題・確認）
パターン2(χ－　)(χ－　)　← 両マイナス
χ²－bχ＋c　（b＞0，c＞0）
ルール：かけてc，たして－b になる負の整数の組を探す
定数項がプラス・χの係数がマイナス → 2数とも－
例）χ²－5χ＋6　→　－2と－3　→　(χ－2)(χ－3)
➡ パターン2 を練習する（例題・類題・確認）
パターン3(χ＋　)(χ－　)　← 符号が逆
χ²＋bχ－c　または　χ²－bχ－c
ルール：定数項がマイナス → 2数は符号が逆（一方＋，一方－）
数の大きい方がχの係数と同じ符号になる
例）χ²＋χ－6　→　＋3と－2　→　(χ＋3)(χ－2)
➡ パターン3 を練習する（例題・類題・確認）
