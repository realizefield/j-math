① こんな問題
📌 道幅問題とは
「縦$12$m，横$16$mの長方形の土地に，$x$mの幅の道を縦横に作って，4つの花壇ア・イ・ウ・エを作ります。
花壇の面積の合計が$140$m²のとき，道幅$x$は何mか？」
緑：花壇　黄：道
② うまい考え方：道を端に寄せる！
📌 アイデア
道を 端に寄せても，花壇の合計面積は変わらない！
なぜ？　花壇は4つに分かれてるけど，合計の縦と横は同じだから 💡
📌 元の図
📌 道を端に寄せた図
道を端に寄せると，花壇は 1つの長方形 になる！
③ 花壇の縦と横は？
📌 道を端に寄せた花壇
花壇の縦は，元の縦から道幅$x$を引いた → $12 - x$
花壇の横は，元の横から道幅$x$を引いた → $16 - x$
↑ 花壇は (12-x) × (16-x) の長方形！
④ 方程式を立てよう
📌 花壇の面積
$(12 - x)(16 - x) = 140$
📌 展開して整理
$(12 - x)(16 - x) = 140$
$192 - 12x - 16x + x^2 = 140$
$192 - 28x + x^2 = 140$
$x^2 - 28x + 52 = 0$
⑤ 解を求めよう
📌 因数分解（少しコツが必要）
$x^2 - 28x + 52 = 0$
たして $-28$，かけて $52$ → $-2$ と $-26$
$(x-2)(x-26) = 0$
$x = 2, 26$
💡 因数分解のコツ
道幅は普通 1m, 2m, 3m, 4m くらい。
だから因数分解で「$-1$ と $\square$」「$-2$ と $\square$」「$-3$ と $\square$」… と試すとよい！
⑥ 解の吟味（とても大事！）
道幅が $26$m？　でも土地は $12$m × $16$m。道は土地より広くなれない！
📌 道幅の条件
道は土地の中に作るので，$0 < x < 12$（短い辺より小さい）
$x = 26$ は不適！　$x = 2$ が答え。
道幅 $x = 2$ m
⑦ 解き方の流れ
📌 道幅問題の手順（5ステップ）
1
道を 端に寄せるイメージで考える
2
花壇の縦・横は $($縦$-x)$，$($横$-x)$
3
面積 = $($縦$-x) \times ($横$-x)$ で 方程式
4
展開・整理して 因数分解 で解く
5
$0 < x < $（短い辺）で吟味
⑧ 「花壇の面積」と「残りの面積」の言い方
📌 同じことを違う言い方
✅ 「花壇の面積の合計が$140$m²」
✅ 「道以外の部分の面積が$140$m²」
✅ 「残りの面積の総和が$140$m²」
どれも 同じ意味！　$($縦$-x)($横$-x) = 140$
📌 まとめ
  • 道は 端に寄せて考える（面積は変わらない）
  • 花壇は $($縦$-x) \times ($横$-x)$ の長方形
  • これを花壇の面積と等しいとおく
  • 展開・整理して因数分解
  • 道幅の条件 $0 < x < $（短い辺）で吟味
  • 道幅は普通 1m〜4m くらいなので，小さい数で因数分解を試す
