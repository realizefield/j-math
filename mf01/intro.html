① 連立方程式とは
📌 連立方程式の意味
χ と ｙ を含む 2つの方程式を組み合わせたもの。
2つの式を同時に満たす χ，ｙ の値を求める。
{
χ＋ｙ＝3……①
χ－ｙ＝1……②
この2つを同時に満たす (χ，ｙ) を求める
② 方程式は天秤のつり合い！
「χ＋ｙ＝3」の天秤　——　左辺と右辺がつり合っている
天秤の左右が「等しい」状態 → 両辺に同じことをしてもつり合いは保たれる
①＋② で「ｙ」を消す！
【ポイント】①の左辺に②の左辺を，①の右辺に②の右辺をそれぞれ足しても天秤のつり合いは変わらない。
ｙ と (－ｙ) が打ち消し合って消え，2χ＝4 → χ＝2 が求まる！
③ ルールを確認しよう
📌 加減法「①＋②」の手順
❶ 一方の文字の係数が ＋と－ の対になっているか確認
❷ 2つの式をたす（①＋②） → その文字が消える
❸ 解いて，求めた値を①か②に代入 → もう一方の値を求める
①＋②でたして消す
「＋a」と「－a」が同じ文字！→ たすと0になる
④ 例で確認しよう
例１）ｙの係数が＋1と－1
{
χ＋ｙ＝3……①
χ－ｙ＝1……②
着目
ｙの係数が ＋1 と －1 → ①＋② でｙが消える
①＋②
2χ＝4　→　χ＝2
代入
χ＝2 を①へ：2＋ｙ＝3　→　ｙ＝1
答え：(χ，ｙ)＝(2，1)
例２）χの係数が－1と＋1
{
－χ＋2ｙ＝4……①
　χ＋3ｙ＝1……②
着目
χの係数が －1 と ＋1 → ①＋② でχが消える
①＋②
5ｙ＝5　→　ｙ＝1
代入
ｙ＝1 を②へ：χ＋3＝1　→　χ＝－2
答え：(χ，ｙ)＝(－2，1)
例３）ｙの係数が＋2と－2
{
3χ＋2ｙ＝12……①
4χ－2ｙ＝ 2……②
着目
ｙの係数が ＋2 と －2 → ①＋② でｙが消える
①＋②
7χ＝14　→　χ＝2
代入
χ＝2 を①へ：6＋2ｙ＝12　→　ｙ＝3
答え：(χ，ｙ)＝(2，3)
📌 まとめ
  • 方程式は天秤のつり合い。①②の天秤を両辺にそれぞれ足してもつり合いは変わらない
  • ①＋②：同じ文字に「＋a」と「－a」があるとき，たすと消える
  • 消えた後は1文字の方程式 → 解いてもう一方の式へ代入
