① 分数を含む連立方程式
📌 分数型の解き方
分数の方程式はそのまま計算しにくい。
→ 分母の 最小公倍数を両辺にかける と整数になり，いつもの加減法で解ける！
分母の最小公倍数を求める
両辺にかけて分数を消す
例１）χ3＋ｙ5＝3　と　χ＋ｙ＝11
{
χ3＋ｙ5＝3……①
χ＋ｙ＝11……②
方針
①の分母：3と5 → lcm＝15。①の両辺に×15
変形
①×15：5χ＋3ｙ＝45　……①'
②×5－①'
(5χ＋5ｙ)－(5χ＋3ｙ)＝55－45　→　2ｙ＝10　→　ｙ＝5
代入
ｙ＝5 を②へ：χ＋5＝11　→　χ＝6
答え：(χ，ｙ)＝(6，5)
例２）χ5＋ｙ2＝3　と　χ＋ｙ＝－3
{
χ5＋ｙ2＝3……①
χ＋ｙ＝－3……②
方針
①の分母：5と2 → lcm＝10。①の両辺に×10
変形
①×10：2χ＋5ｙ＝30　……①'
②×2－①'
(2χ＋2ｙ)－(2χ＋5ｙ)＝－6－30　→　－3ｙ＝－36　→　ｙ＝12
代入
ｙ＝12 を②へ：χ＋12＝－3　→　χ＝－15
答え：(χ，ｙ)＝(－15，12)
例３）2χ3＋ｙ＝4　と　χ＋3ｙ＝9
{
2χ3＋ｙ＝4……①
χ＋3ｙ＝9……②
方針
①の分母：3 → 両辺に×3
変形
①×3：2χ＋3ｙ＝12　……①'
①'－②
(2χ＋3ｙ)－(χ＋3ｙ)＝12－9　→　χ＝3
代入
χ＝3 を②へ：3＋3ｙ＝9　→　ｙ＝2
答え：(χ，ｙ)＝(3，2)
② 小数を含む連立方程式
📌 小数型の解き方
小数点以下の桁数に合わせて ×10 や ×100 をして整数にする！
整数になったら，いつも通りの加減法で解ける。
小数1桁 → ×10
小数2桁 → ×100
例４）0.5χ＋0.3ｙ＝－0.7　と　χ＋ｙ＝－1
{
χ＋ｙ＝－1……①
0.5χ＋0.3ｙ＝－0.7……②
方針
②の小数1桁 → 両辺に×10
変形
②×10：5χ＋3ｙ＝－7　……②'
①×5－②'
(5χ＋5ｙ)－(5χ＋3ｙ)＝－5－(－7)　→　2ｙ＝2　→　ｙ＝1
代入
ｙ＝1 を①へ：χ＋1＝－1　→　χ＝－2
答え：(χ，ｙ)＝(－2，1)
例５）χ＋ｙ＝5　と　0.3χ＋0.7ｙ＝2.3
{
χ＋ｙ＝5……①
0.3χ＋0.7ｙ＝2.3……②
方針
②の小数1桁 → 両辺に×10
変形
②×10：3χ＋7ｙ＝23　……②'
②'－①×3
(3χ＋7ｙ)－(3χ＋3ｙ)＝23－15　→　4ｙ＝8　→　ｙ＝2
代入
ｙ＝2 を①へ：χ＋2＝5　→　χ＝3
答え：(χ，ｙ)＝(3，2)
例６）χ＋ｙ＝7　と　0.25χ＋0.33ｙ＝2
{
χ＋ｙ＝7……①
χ4＋ｙ3＝2……②
方針
②の分母：4と3 → lcm＝12。両辺に×12
変形
②×12：3χ＋4ｙ＝24　……②'
②'－①×3
(3χ＋4ｙ)－(3χ＋3ｙ)＝24－21　→　ｙ＝3
代入
ｙ＝3 を①へ：χ＋3＝7　→　χ＝4
答え：(χ，ｙ)＝(4，3)
📌 まとめ
  • 分数型：分母のLCMを両辺にかけて整数にしてから解く
  • 小数型：×10 や ×100 で整数にしてから解く
  • 整数になったら，あとはいつもの加減法（①＋②，①－②，係数そろえ）で解ける
