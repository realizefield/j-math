公式の早見表
公式①
(χ＋a)(χ＋b)＝χ²＋(a＋b)χ＋ab　← たして・かける
公式②
(χ＋a)²＝χ²＋2aχ＋a²　← 前²＋前×後×2＋後²
公式③
(χ＋a)(χ－a)＝χ²－a²　← 真ん中が消える
🔑 どの公式を使うか見分けるコツ
1
まず形を確認する
同じ文字が2回 → 公式②か③を疑う
(　＋　)(　＋　) で同じ符号 → 公式①か②
(　＋　)(　－　) で同じ前 → 公式③
2
公式②と①の見分け方
後ろ2つが同じ数 → 公式②（例：(χ＋3)²）
後ろ2つが違う数 → 公式①（例：(χ＋3)(χ＋5)）
3
公式③の決め手
前が同じで符号だけ違う → 必ず公式③（例：(χ＋4)(χ－4)）
迷ったら「前が同じで符号が逆」→公式③、「2乗」→公式②、それ以外→公式① の順に確認しよう！
