① 「式の値」とは？
📌 文字に値を代入して計算する
たとえば文字式 $x^2 + 3x$ で，$x = 2$ のとき：
$2^2 + 3 \times 2 = 4 + 6 = 10$
これが「式の値」です。
SQ08では，$x$ にルートを含む値が入る ケースを学びます。
📌 たとえばこんな問題
$x = \sqrt{3} + 1$ のとき，$x^2 - 2x$ の値を求めなさい。
一見むずかしそうに見えますが，SQ07までの計算 ができていれば必ず解けます！
② 基本：直接代入する
📌 基本のやり方
$x$ にそのままルートを含む値を代入し，計算する
📌 例：$x = \sqrt{3} + 1$ のとき $x^2 - 2x$
まず $x^2$ を計算：
$x^2 = (\sqrt{3}+1)^2 = 3 + 2\sqrt{3} + 1 = 4 + 2\sqrt{3}$
次に $2x$：
$2x = 2(\sqrt{3}+1) = 2\sqrt{3} + 2$
引き算：
$x^2 - 2x = (4+2\sqrt{3}) - (2\sqrt{3}+2) = 2$
$\sqrt{3}$ の部分が うまく消える ことが多い！　答えが整数になりやすいのが特徴。
③ 応用：式を変形してから代入
📌 直接代入だと大変な場合
$x^2 + 4x$ に $x = \sqrt{5} - 2$ を代入するとき，まともに計算すると面倒。
でも，式を先に変形すると楽 になることがあります。
💡 テクニック：因数分解して代入
$x^2 + 4x = x(x + 4)$ と因数分解できる。
$x = \sqrt{5} - 2$ なので $x + 4 = \sqrt{5} + 2$
積を計算：$(\sqrt{5}-2)(\sqrt{5}+2) = (\sqrt{5})^2 - 2^2 = 5 - 4 = 1$
和と差の積(SQ07)を利用して，あっという間に答えが出る！
④ よく使う公式
📌 平方の和・差の公式
$x^2 + y^2 = (x+y)^2 - 2xy$
$x^2 - y^2 = (x+y)(x-y)$
📌 なぜ便利？
$x = \sqrt{2}+1$，$y = \sqrt{2}-1$ のとき，$x^2 + y^2$ を求める場合：
直接計算すると面倒だが，先に $x+y$ と $xy$ を求めれば楽！
$x + y = 2\sqrt{2}$，$xy = (\sqrt{2}+1)(\sqrt{2}-1) = 2-1 = 1$
$x^2 + y^2 = (2\sqrt{2})^2 - 2 \times 1 = 8 - 2 = 6$
$x+y$ や $xy$ が シンプルな値 になるように $x$，$y$ が設定されていることが多い！
⑤ 高認でよく出るパターン
📌 たとえば $x = 3+\sqrt{5}$ のとき $x^2 - 6x + 4$
まず $x - 3 = \sqrt{5}$ になることに注目！
両辺を2乗：$(x-3)^2 = 5$ → $x^2 - 6x + 9 = 5$ → $x^2 - 6x = -4$
よって $x^2 - 6x + 4 = -4 + 4 = 0$
💡 テクニック：$x - \text{整数} = \sqrt{\phantom{x}}$ の形を作る
式の中の 「2次の項」と「1次の項」のセット をうまく使うと，ルートを消せることがある！
高認の入試によく出るので，慣れておこう。
⑥ 例で確認しよう
例)$x = 2\sqrt{3}$ のとき $x^2 - 7$ の値
代入する
$x^2 = (2\sqrt{3})^2$
展開
$(2)^2 \times (\sqrt{3})^2 = 4 \times 3 = 12$
引き算
$12 - 7 = 5$
$x^2 - 7 = 5$
📌 まとめ
  • SQ08は SQ07までの総まとめ 的な単元
  • 基本は そのまま代入 して計算
  • 大変なときは 式を変形 してから代入
  • $(\sqrt{a}+b)(\sqrt{a}-b) = a-b^2$ (和と差の積)が大活躍
  • $x^2 + y^2 = (x+y)^2 - 2xy$ も覚えておく
  • 高認頻出：$x - \text{整数} = \sqrt{\phantom{x}}$ の形に注目！
