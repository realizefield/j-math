QE02

$(x+a)^2 = b$ の形
かたまりを作って解く

④確認
① 前回(QE01)のおさらい
📌 $x^2 = a$ の形は解けるようになった
$x^2 = 5$ → $x = \pm\sqrt{5}$
$4x^2 = 9$ → $x = \pm\dfrac{3}{2}$
では，$(x+1)^2 = 7$ のような問題はどう解く？
② $(x+1)^2 = 7$ をどう解く？
📌 アイデア：$x+1$ を「かたまり」として見る
$x+1$ を $A$ という1つの文字 に置きかえてみよう。
$A^2 = 7$ になる！　これなら QE01 で習った形！
$A^2 = 7$ → $A = \pm\sqrt{7}$
📌 $A$ を $x+1$ に戻す
$A = x + 1$ だったので，
$x + 1 = \pm\sqrt{7}$
$-1$ を移項して：$x = -1 \pm\sqrt{7}$
$x = -1 \pm\sqrt{7}$
$A$ と置きかえて考える と，知っている形にできる！
③ 慣れたら一気に解こう
📌 $(x+a)^2 = b$ の解き方
$(x+a)^2 = b$ → $x+a = \pm\sqrt{b}$ → $x = -a \pm\sqrt{b}$
📌 慣れたら $A$ なしで一気に！
$(x+1)^2 = 7$ を見たら…
すぐに「$x+1 = \pm\sqrt{7}$」と書く！
あとは移項して $x = -1 \pm\sqrt{7}$
$(x+1)^2 = 7$ → $x+1 = \pm\sqrt{7}$ → $x = -1 \pm\sqrt{7}$
④ 整数の解になることもある
📌 $b$ が平方数のとき
$(x+1)^2 = 4$ を解くと：
$x + 1 = \pm 2$
$x = -1 \pm 2$
つまり $x = -1 + 2 = 1$ と $x = -1 - 2 = -3$
答えは 2つの整数！
$x = 1, -3$
$\pm$ の答えが整数になるときは，$+$ と $-$ を別々に計算 して，2つの値を書く。
⑤ マイナス記号にも注意
📌 $(x-1)^2 = 5$ の場合
かっこの中が $x - 1$ なので…
$x - 1 = \pm\sqrt{5}$
$-1$ を移項（プラスにして反対側へ）：
$x = 1 \pm\sqrt{5}$
$(x \color{red}{-} 1)^2$ の場合，移項したら $+1$ になる！　符号のミスに注意！
⑥ 移項が必要な場合
📌 $(x+1)^2 - 16 = 0$ の形
まず $-16$ を移項：$(x+1)^2 = 16$
あとは普通に：$x + 1 = \pm 4$
$x = -1 \pm 4$
つまり $x = 3, -5$
💡 解き方の流れ
① $(x+a)^2 = b$ の形に整える（移項する）
② かたまりに $\pm\sqrt{\phantom{x}}$ をつけて外す
③ $x = $ の形にする（移項）
④ 答えが整数2つになる場合は $+$ と $-$ を別々に計算
⑦ 例で確認しよう
例１)$(x+2)^2 = 9$ を解く
ルートを取る
$x + 2 = \pm 3$
移項
$x = -2 \pm 3$
$+$ と $-$ を計算
$x = -2+3 = 1$，$x = -2-3 = -5$
$x = 1, -5$
例２)$(x-3)^2 = 11$ を解く
ルートを取る
$x - 3 = \pm\sqrt{11}$
移項
$x = 3 \pm\sqrt{11}$　（$\sqrt{11}$ は簡単化できないのでそのまま）
$x = 3 \pm\sqrt{11}$
📌 まとめ
  • $(x+a)^2 = b$ の形は，かたまりとして見る
  • $(x+a)^2 = b$ → $x+a = \pm\sqrt{b}$ → $x = -a \pm\sqrt{b}$
  • $b$ が平方数なら，整数2つ が解
  • $b$ が平方数じゃなければ，ルートが残る
  • $(x+a)^2 = b$ の形になるよう まず移項 する場合もある
  • $\sqrt{b}$ は 簡単化できるか もチェック！(SQ04)
