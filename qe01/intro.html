① 二次方程式とは？
📌 これまでとちがう新しい方程式
一次方程式：$3x + 5 = 11$ のように $x$ が 1乗 の方程式
二次方程式：$x^2 - x - 6 = 0$ のように $x$ の 2乗 が出てくる方程式
「$x^2$ がある」のが特徴です！
📌 二次方程式の特徴
一次方程式の解は普通 1個（$x = 3$ など）
二次方程式の解は普通 2個（$x = -2, 3$ など）
なぜ2個になるのか？　その理由はこのあとわかります！
② 二次方程式の3つの解き方
📌 これから学ぶ流れ
二次方程式には 3通りの解き方 があります：
① $x^2 = a$ や $(x+a)^2 = b$ の形（QE01・QE02）
② 因数分解で解く（QE03・QE04）
③ 解の公式（QE05・QE06）
今回(QE01)はいちばん基本の $x^2 = a$ の形 から始めます。
③ $x^2 = a$ を解こう
📌 基本の考え方
$x^2 = a$ ならば $x = \pm\sqrt{a}$
($a \geq 0$ のとき)
📌 平方根のおさらい(SQ01)
$x^2 = 4$ になる $x$ を考えると…
$2^2 = 4$ なので $x = 2$
でも $(-2)^2 = 4$ でもあるので $x = -2$ も解！
だから $x = \pm 2$ と書きます。
$x^2 = 4$ → $x = \pm 2$
「$\pm$」は「プラスマイナス」と読む。解が2つある ことをまとめて表す記号！
④ 答えにルートが残るとき
📌 たとえば $x^2 = 5$ の場合
$5$ は平方数ではないので，2乗して $5$ になるきれいな数はない。
でも，平方根の記号 $\sqrt{\phantom{x}}$ を使えば書ける！
$x^2 = 5$ → $x = \pm\sqrt{5}$
$\sqrt{5}$ がそのまま答えとして OK！　無理に近似値にしなくてよい。
⑤ 係数がある場合
📌 たとえば $4x^2 = 9$
まず両辺を $4$ でわって，$x^2 = \dfrac{9}{4}$ の形にする。
次に，$x = \pm\sqrt{\dfrac{9}{4}} = \pm\dfrac{3}{2}$
$4x^2 = 9$ → $x^2 = \dfrac{9}{4}$ → $x = \pm\dfrac{3}{2}$
📌 移項が必要な場合
$4x^2 - 49 = 0$ のような形は，まず $-49$ を移項 して $4x^2 = 49$ の形にしてから解く。
⑥ 例で確認しよう
例１)$x^2 = 9$ を解く
考え方
$\pm\sqrt{9}$ を求める
計算
$\sqrt{9} = 3$ なので $x = \pm 3$
$x = \pm 3$（つまり $x = 3$ と $x = -3$）
例２)$x^2 = 2$ を解く
確認
$2$ は平方数ではない → ルートが残る
答え
$x = \pm\sqrt{2}$
$x = \pm\sqrt{2}$
例３)$x^2 = 6$ を解く
確認
$6$ も平方数ではない
答え
$x = \pm\sqrt{6}$
$x = \pm\sqrt{6}$
📌 まとめ
  • 二次方程式は $x^2$ が出てくる方程式
  • 解は普通 2つ（$\pm$ で表す）
  • $x^2 = a$ ならば $x = \pm\sqrt{a}$
  • $a$ が平方数なら整数の解，そうでなければ $\sqrt{\phantom{x}}$ が残る
  • 係数がある場合は，まず $x^2 = \square$ の形 にする
  • $\square$ の中が分数になることもある
