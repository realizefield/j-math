① 代入法とは
📌 代入法の意味
一方の方程式を「χ＝…」または「ｙ＝…」の形に変えて，
それをもう一方の式に代入（埋め込む）して1文字の方程式にする方法。

どちらかの式が「ｙ＝…」や「χ＝…」の形をしていたら 代入法が超便利！
📌 代入法の手順
❶ 一方の式を「ｙ＝…」か「χ＝…」の形にする
❷ その式をもう一方の式に代入（ｙやχの部分に「式ごと」入れる）
❸ 1文字の方程式を解く
❹ 求めた値を代入して，もう一方の値を求める
「ｙ＝…」や「χ＝…」があれば
代入法が便利！
係数が整数の普通の式
→ 加減法でもOK
② 例で確認しよう
例１）「ｙ＝…」の形が既にある（そのまま代入）
{
ｙ＝2χ－1……①
3χ＋ｙ＝9……②
方針
①が「ｙ＝…」の形 → ②のｙに①をそのまま代入
代入
②に ｙ＝2χ－1 を代入：
3χ＋(2χ－1)＝9 → 5χ－1＝9 → 5χ＝10 → χ＝2
代入
χ＝2 を①へ：ｙ＝2×2－1＝3
答え：(χ，ｙ)＝(2，3)
例２）「χ＝…」の形が既にある（そのまま代入）
{
χ＝3ｙ＋1……①
2χ＋ｙ＝9……②
方針
①が「χ＝…」の形 → ②のχに①をそのまま代入
代入
②に χ＝3ｙ＋1 を代入：
2×(3ｙ＋1)＋ｙ＝9 → 6ｙ＋2＋ｙ＝9 → 7ｙ＝7 → ｙ＝1
代入
ｙ＝1 を①へ：χ＝3×1＋1＝4
答え：(χ，ｙ)＝(4，1)
例３）変形してから代入（一方を「ｙ＝…」に変形）
{
χ＋ｙ＝5……①
2χ－ｙ＝4……②
変形
①を変形 → ｙ＝5－χ　……①'
代入
②に ｙ＝5－χ を代入：
2χ－(5－χ)＝4 → 2χ－5＋χ＝4 → 3χ＝9 → χ＝3
代入
χ＝3 を①'へ：ｙ＝5－3＝2
答え：(χ，ｙ)＝(3，2)
例４）変形してから代入（2χ＋ｙ＝7 → ｙ＝7－2χ）
{
2χ＋ｙ＝7……①
3χ－2ｙ＝0……②
変形
①を変形 → ｙ＝7－2χ　……①'
代入
②に ｙ＝7－2χ を代入：
3χ－2×(7－2χ)＝0 → 3χ－14＋4χ＝0 → 7χ＝14 → χ＝2
代入
χ＝2 を①'へ：ｙ＝7－4＝3
答え：(χ，ｙ)＝(2，3)
例５）変形してから代入（χ＋2ｙ＝8 → χ＝8－2ｙ）
{
χ＋2ｙ＝8……①
3χ－ｙ＝3……②
変形
①を変形 → χ＝8－2ｙ　……①'
代入
②に χ＝8－2ｙ を代入：
3×(8－2ｙ)－ｙ＝3 → 24－6ｙ－ｙ＝3 → 7ｙ＝21 → ｙ＝3
代入
ｙ＝3 を①'へ：χ＝8－6＝2
答え：(χ，ｙ)＝(2，3)
📌 まとめ
  • 「ｙ＝…」や「χ＝…」の形があれば → 代入法が便利
  • なければ①を変形して「ｙ＝…」か「χ＝…」にしてから代入
  • 代入するとき，式はかっこを忘れずに：ｙ＝(2χ－1)
  • 加減法でも代入法でも，答えは必ず同じ！使いやすい方を選ぼう
