こんな計算、どうやる？

2.3 ＋ 1.45 を計算してみましょう。

❌ まちがい　— 右端を揃えた場合
2.3_
+1.45
3.??

小数点の位置がズレると正しく計算できない！

⭕ 正しい　— 小数点を縦に揃える
2.30
+1.45
3.75

2.3 を 2.30 と考えて桁を揃える！

小数点を揃えるコツ
STEP 1　小数点を縦にピタッと揃える

筆算を書くとき、小数点の位置を必ず縦に揃える。整数部・小数部がずれないようにする。

STEP 2　桁が足りない部分は 0 で補う

2.3 と 1.45 を筆算にするとき、2.3 の小数第2位がないので 2.30 として考える。
※ 0 を補っても数の大きさは変わらない！

STEP 3　整数のたし算・ひき算と同じように計算し、小数点をそのまま下ろす

筆算で計算して、答えの小数点は揃えた位置からまっすぐ下ろす。

たし算の筆算　例）2.3 ＋ 1.45
くり上がり
1
2.30
+1.45
3.75
  1. 2.3 → 2.30 と書く
  2. 小数点を縦に揃える
  3. 0＋5＝5（小数第2位）
  4. 3＋4＝7（小数第1位）
  5. 2＋1＝3（整数の位）
  6. 答え：3.75
ひき算の筆算　例）3.5 − 1.28
3.50
−1.28
2.22
  1. 3.5 → 3.50 と書く
  2. 小数点を縦に揃える
  3. 0−8 は借りてきて 10−8＝2
  4. 4（借りた後）−2＝2（小数第1位）
  5. 3−1＝2（整数の位）
  6. 答え：2.22
〈重要〉小数の計算ルール
① 小数点を縦に揃える
② 空いた桁を 0 で補う
③ 整数と同じように計算

答えの小数点は揃えた位置からまっすぐ下ろす！

📌 まとめ
  • 筆算では 小数点を縦にピタッと揃える
  • 桁が足りない部分は 0 を補って 考える（2.3 → 2.30）
  • 計算は整数と同じ。答えの小数点は そのまま真下に下ろす
