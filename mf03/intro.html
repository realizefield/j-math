① なぜ係数をそろえるの？
📌 係数をそろえる理由
①＋② や ①－② は「消したい文字の係数が同じ大きさ」のときだけ使える。
係数が違う場合は，両辺に同じ数をかけて係数をそろえてから加減する。

方程式の両辺に同じ数をかけても，方程式の解は変わらない！
消したい文字の係数の
最小公倍数を求める
両辺に整数をかけて
係数をそろえる
② 例で確認しよう
例１）χの係数をそろえる（3と2 → 6）
{
3χ－5ｙ＝4……①
2χ＋3ｙ＝9……②
方針
χを消去したい → χの係数LCM：lcm(3,2)＝6 → ①×2，②×3
変形
①×2：6χ－10ｙ＝8　　②×3：6χ＋9ｙ＝27
①×2－②×3
(6χ－10ｙ)－(6χ＋9ｙ)＝8－27　→　－19ｙ＝－19　→　ｙ＝1
代入
ｙ＝1 を②へ：2χ＋3＝9　→　2χ＝6　→　χ＝3
答え：(χ，ｙ)＝(3，1)
例２）ｙの係数をそろえる（2と3 → 6）
{
9χ＋2ｙ＝24……①
17χ－3ｙ＝25……②
方針
ｙを消去したい → lcm(2,3)＝6 → ①×3，②×2
変形
①×3：27χ＋6ｙ＝72　　②×2：34χ－6ｙ＝50
①×3＋②×2
ｙの符号が逆なので足す：61χ＝122　→　χ＝2
代入
χ＝2 を①へ：18＋2ｙ＝24　→　ｙ＝3
答え：(χ，ｙ)＝(2，3)
例３）ｙの係数をそろえる（6と7 → 42）
{
5χ＋6ｙ＝3……①
4χ＋7ｙ＝－2……②
方針
ｙを消去 → lcm(6,7)＝42 → ①×7，②×6
変形
①×7：35χ＋42ｙ＝21　　②×6：24χ＋42ｙ＝－12
①×7－②×6
11χ＝33　→　χ＝3
代入
χ＝3 を①へ：15＋6ｙ＝3　→　ｙ＝－2
答え：(χ，ｙ)＝(3，－2)
例４）ｙの係数をそろえる（3と2 → 6）
{
2χ＋3ｙ＝－1……①
3χ＋2ｙ＝ 1……②
方針
ｙを消去 → lcm(3,2)＝6 → ①×2，②×3
変形
①×2：4χ＋6ｙ＝－2　　②×3：9χ＋6ｙ＝3
①×2－②×3
(4χ＋6ｙ)－(9χ＋6ｙ)＝－2－3　→　－5χ＝－5　→　χ＝1
代入
χ＝1 を①へ：2＋3ｙ＝－1　→　ｙ＝－1
答え：(χ，ｙ)＝(1，－1)
📌 まとめ
  • 係数が違う → 両辺に整数をかけて係数をそろえる
  • 係数の最小公倍数（LCM）を求めて，それぞれの式にかける
  • そろったら ①±② でその文字を消去 → 1文字の方程式へ
