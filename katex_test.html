KaTeX 動作確認ページ

このページで数式がきれいに表示されていれば、KaTeX は正常に動作しています。

平方根

$\sqrt{2}$、$\sqrt{3}$、$\sqrt{5}$ の値を覚えよう。

$\sqrt{12} = 2\sqrt{3}$

$$\sqrt{50} + \sqrt{18} = 5\sqrt{2} + 3\sqrt{2} = 8\sqrt{2}$$

分数と指数

$\dfrac{1}{2} + \dfrac{1}{3} = \dfrac{5}{6}$

$$x^2 + 2x + 1 = (x+1)^2$$

解の公式

$$x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

入り組んだ式

$$\sqrt{\dfrac{3}{2}} = \dfrac{\sqrt{6}}{2}$$

$$(\sqrt{5} + \sqrt{2})(\sqrt{5} - \sqrt{2}) = 5 - 2 = 3$$

数式がテキストのまま（$\sqrt{2}$のように）表示されている場合、KaTeX の読み込みに失敗しています。
