① 因数分解とは
📌 因数分解の意味
多項式を，いくつかの式の積（かけ算）の形に変えることを「因数分解」といいます。
展開の逆の操作です。
χ²＋5χ＋6　→　(χ＋2)(χ＋3)
② 公式Ⅰ：平方の差 χ²－a²
📌 因数分解公式Ⅰ（展開③の逆）
χ²－a²＝(χ＋a)(χ－a)
○²－□² の形を見つけたらこの公式！
🤔 なぜこうなるの？
EF03の公式③を「逆向き」に使っているだけ！
1
EF03で学んだ公式③（和と差の積）を思い出そう
(χ＋a)(χ－a) ＝ χ²－a²　← 展開の公式
2
因数分解はこれを右から左に読むだけ！
χ²－a² ＝ (χ＋a)(χ－a)　← 因数分解の公式
3
「２乗の差」を見つけたら，何と何の2乗かを探す
χ²－25 → χ² と 5² の差 → (χ＋5)(χ－5)
因数分解は展開の「逆操作」。展開の公式を右から左に読めばOK！
例１）χ²－25 を因数分解する
形
χ²－25＝χ²－5²　→ a＝5
公式Ⅰ
χ²－5²＝(χ＋5)(χ－5)
(χ＋5)(χ－5)
例２）9χ²－16 を因数分解する（前がχ²以外）
形
9χ²＝(3χ)²，16＝4²　→ a＝3χ，b＝4
公式Ⅰ
(3χ)²－4²＝(3χ＋4)(3χ－4)
(3χ＋4)(3χ－4)
③ 公式Ⅱ：たして・かける因数分解 χ²＋bχ＋c
📌 因数分解公式Ⅱ（展開①の逆）
χ²＋(a＋b)χ＋ab＝(χ＋a)(χ＋b)
かけてc，たしてb となる2数 a,b を探す！
🤔 なぜ「かけて・たして」で見つかるの？
EF01の展開を逆向きに考えよう！
1
EF01で学んだ展開の公式を思い出そう
(χ＋a)(χ＋b) ＝ χ²＋(a＋b)χ＋ab
2
χの係数は a＋b（たした値），定数項は ab（かけた値）
だから逆に，「かけてc，たしてb」になる2数を探せばいい！
3
例：χ²＋5χ＋6　→　かけて6，たして5になる2数は？
1×6＝6，1＋6＝7　✗　　2×3＝6，2＋3＝5　✅
→ (χ＋2)(χ＋3)
符号のルール：定数項がプラスで係数がマイナス → 2数とも負。定数項がマイナス → 2数は符号が逆（大きい方が係数と同じ符号）。
STEP1
かけてc になる数の組を列挙
STEP2
その中でたしてb になる組を選ぶ
例３）χ²＋6χ＋8 を因数分解する
STEP1
かけて8：(1,8)，(2,4)，(-1,-8)，(-2,-4)
STEP2
たして6 になるのは → 2と4
(χ＋2)(χ＋4)
例４）χ²－9χ＋18 を因数分解する（両方マイナス）
ポイント
かけて＋，たしてマイナス → 2数とも負！
探す
かけて18，たして－9 → －3と－6
(χ－3)(χ－6)
例５）χ²－x－6 を因数分解する（定数項がマイナス）
ポイント
かけてマイナス → 2数は符号が逆
探す
かけて－6，たして－1 → 2と－3
(χ＋2)(χ－3)
📌 まとめ
  • χ²－a²＝(χ＋a)(χ－a)
  • χ²＋bχ＋c＝(χ＋p)(χ＋q)　「かけてc，たしてb」
  • 定数項がプラス・χの係数がマイナス → 2数とも負
  • 定数項がマイナス → 2数は符号が逆
