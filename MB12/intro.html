① 計算の順番（全部のまとめ）

今まで学んだ計算の優先順位を整理しましょう。

📏 計算の優先順位（高い順）
1
指数（累乗）
(−3)²、2³ などを最初に計算
2
カッコの中　( ) → { }
内側の( )から順に計算
3
× と ÷
＋−より先に計算
4
＋ と −
最後に計算
② 指数を先に計算する
例） (−3)² − 2³ − 7
1
指数を先に計算
(−3)2＝9　　23＝8
2
代入して計算
9 − 8 − 7 ＝ −6
⚠️ 注意！ −6² と (−6)² は違う
−62
＝ −(6×6) ＝ −36
(−6)2
＝ (−6)×(−6) ＝ ＋36
③ 中カッコ { } の計算

中カッコ { } は、内側の ( ) を先に計算してから { } を計算します。

計算順：（　）の中 → ×÷ → ＋− → {　} → 全体
例） (−3)² − {6 − (2×3 − 7)}
1
指数を計算
(−3)2 ＝ 9
2
( )の中を先に：2×3−7
2×3 ＝ 6　→　6−7 ＝ −1
3
{ }の中を計算：6−(−1)
6 − （−1） ＝ 6 ＋ 1 ＝ 7
4
全体を計算
9 − 7 ＝ 2
