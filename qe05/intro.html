① こんな二次方程式，どう解く？
📌 たとえば $2x^2 - 3x - 1 = 0$
QE03 の「たして・かける」で因数分解しようとしても…
たして $-\dfrac{3}{2}$ かけて $-\dfrac{1}{2}$？　すっきりしない！
実は，こういう 因数分解できない式 を解くための 魔法の公式 があります 🧙
② 解の公式とは
📌 解の公式
$ax^2 + bx + c = 0$ のとき
$x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
📌 大事なこと
この公式は どんな二次方程式でも解ける！
ただし，因数分解できる場合は QE03・QE04 のほうが楽 なので，
解の公式は 因数分解できないとき に使います。
解の公式は 必ず暗記！　高認や入試で何度も使う！
💡 これは 覚えなくてもOK ！　でも「なぜこの公式になるの？」を知ると面白い！
$(x+a)^2 = b$ の形にする「平方完成」がポイントです 🌟
1
$$ax^2 + bx + c = 0$$ から出発
↓ 両辺を $a$ でわる
2
$$x^2 + \dfrac{b}{a}x + \dfrac{c}{a} = 0$$
↓ $\left(\dfrac{b}{2a}\right)^2$ を足して引く（平方完成のテクニック）
3
$$x^2 + \dfrac{b}{a}x + \left(\dfrac{b}{2a}\right)^2 - \left(\dfrac{b}{2a}\right)^2 + \dfrac{c}{a} = 0$$
↓ 前半3つは $(x+a)^2$ の形にできる！
4
$$\left(x + \dfrac{b}{2a}\right)^2 - \dfrac{b^2}{4a^2} + \dfrac{c}{a} = 0$$
↓ 残りの項を整理（通分）
5
$$\left(x + \dfrac{b}{2a}\right)^2 = \dfrac{b^2 - 4ac}{4a^2}$$
↓ ルートを取る（QE02 の形！）
6
$$x + \dfrac{b}{2a} = \pm\dfrac{\sqrt{b^2 - 4ac}}{2a}$$
↓ $\dfrac{b}{2a}$ を移項
7
$$x = -\dfrac{b}{2a} \pm \dfrac{\sqrt{b^2 - 4ac}}{2a}$$
↓ 分母が同じなのでまとめる
$$x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$ 🎉
つまり，解の公式は $(x+a)^2 = b$ の形に変形して解いた結果 なんです！
③ $a, b, c$ の見つけ方
📌 標準形 $ax^2 + bx + c = 0$ と比べる
たとえば $2x^2 - 3x - 1 = 0$ なら：
$a = 2$（$x^2$ の係数）
$b = -3$（$x$ の係数 ← マイナスも忘れずに！）
$c = -1$（定数項 ← マイナスも忘れずに！）
式$a$$b$$c$
$2x^2 - 3x - 1 = 0$$2$$-3$$-1$
$3x^2 + 5x + 2 = 0$$3$$5$$2$
$x^2 - 4x - 7 = 0$$1$$-4$$-7$
$b$ と $c$ の 符号（プラス・マイナス）を間違えやすい！　慎重に！
④ 解の公式を使ってみよう
📌 例：$2x^2 - 3x - 1 = 0$ を解く
$a = 2$，$b = -3$，$c = -1$ を公式に代入！
$x = \dfrac{-(-3) \pm \sqrt{(-3)^2 - 4 \times 2 \times (-1)}}{2 \times 2}$
📌 中身を計算
$-(-3) = 3$
$(-3)^2 = 9$
$4 \times 2 \times (-1) = -8$
$b^2 - 4ac = 9 - (-8) = 9 + 8 = 17$
$2a = 4$
$x = \dfrac{3 \pm \sqrt{17}}{4}$
$x = \dfrac{3 \pm \sqrt{17}}{4}$
⑤ 計算のコツ
💡 ステップで考えよう
① $a, b, c$ を見つける（符号に注意！）
② $-b$ を計算
③ $b^2$ を計算
④ $4ac$ を計算
⑤ $b^2 - 4ac$ を計算（ルートの中身）
⑥ $2a$ を計算（分母）
⑦ 全部組み合わせる
$-b$ と $b^2$ は別の計算！　$b = -3$ なら $-b = 3$，$b^2 = 9$。
⑥ ルートが整数になるとき
📌 例：$5x^2 + 7x + 2 = 0$
$a = 5$，$b = 7$，$c = 2$
$b^2 - 4ac = 49 - 40 = 9$
$\sqrt{9} = 3$（整数！）
$x = \dfrac{-7 \pm 3}{10}$
プラスとマイナスを別々に計算：
$x = \dfrac{-7+3}{10} = -\dfrac{4}{10} = -\dfrac{2}{5}$
$x = \dfrac{-7-3}{10} = -\dfrac{10}{10} = -1$
$x = -\dfrac{2}{5}, -1$
$\sqrt{\square}$ の中が 平方数 になったら，整数や分数の解になる！
⑦ 例で確認しよう
例)$x^2 + 3x + 1 = 0$ を解く
$a, b, c$
$a = 1$，$b = 3$，$c = 1$
$b^2 - 4ac$
$3^2 - 4 \times 1 \times 1 = 9 - 4 = 5$
公式へ代入
$x = \dfrac{-3 \pm \sqrt{5}}{2}$
$x = \dfrac{-3 \pm \sqrt{5}}{2}$
📌 まとめ
  • 解の公式：$x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}$
  • 因数分解できないとき に使う
  • $ax^2 + bx + c = 0$ から $a, b, c$ を見つける（符号に注意！）
  • 計算の順序：$-b$ → $b^2$ → $4ac$ → $b^2-4ac$ → $2a$
  • $\sqrt{\phantom{x}}$ の中が平方数なら，整数や分数の解になる
  • 解の公式は 必ず暗記！
